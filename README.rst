Skill Switch

Run
---

To run the project on Linux, you will need to install `SFML
<https://www.sfml-dev.org/>`_, gcc/clang and cmake.

Then ``git clone`` the project, create a build folder and do the following:

.. code-block:: shell

	make
	./build/run
